Buat sebuah web-based tool menggunakan PHP prosedural dan Bootstrap 5 yang memiliki fungsi:

🎯 Tujuan:
Membuat tool yang dapat melakukan:

Code obfuscation (mengacak nama variabel, menghapus whitespace/komentar, encoding string)

Chunk-based obfuscation & encryption (memecah file menjadi potongan terenkripsi)

Membuat loader runtime untuk menjalankan chunk hasil enkripsi

🖥️ Antarmuka (UI Web dengan Bootstrap 5):
Buat tampilan yang mobile & desktop friendly menggunakan Bootstrap Grid

Halaman utama memiliki tab/menu navigasi:

Upload File

Obfuscate

Chunk Obfuscate

Generate Loader

Output / Download Result

Gunakan warna netral, ikon untuk tombol (Font Awesome), dan animasi ringan

Progress bar atau status ketika proses berlangsung

🔧 Fungsi-Fungsi Wajib:
✅ 1. Code Obfuscator
Rename semua variabel, fungsi, dan nama class secara acak

Hapus komentar dan whitespace

Encode string literal dengan base64_encode(), decode saat runtime

Simpan hasil sebagai .obf.php

✅ 2. Chunk-Based Obfuscator
Pisahkan file PHP menjadi beberapa bagian (fungsi atau ukuran max 5KB per chunk)

Obfuscate masing-masing chunk

Enkripsi tiap chunk dengan AES-256-CBC (gunakan openssl_encrypt())

Simpan ke /output/chunks/*.php.enc

Buat file chunks.map.json berisi:

json
Copy
Edit
{
  "chunks": [
    {"file": "chunk_01.php.enc", "order": 1, "checksum": "abc..."},
    ...
  ],
  "key_hint": "manual or license-based"
}
✅ 3. Loader Generator
Buat file loader.php yang:

Membaca chunks.map.json

Mendekripsi tiap chunk

Mengeksekusi dengan eval()

Bisa diintegrasikan dengan verifikasi lisensi (opsional)

⚙️ Teknologi:
PHP 7.4+ (tanpa framework)

Bootstrap 5 (CDN atau lokal)

Tidak menggunakan database

Semua file disimpan lokal, no external API

Log hasil proses ke /logs/

Checkbox opsi:

Mode debug (lihat hasil sebelum dan sesudah)

Link download hasil .obf.php

Download .map.json, file chunk terenkripsi, dan loader.php
